### 문제 설명

<p>정수 $N$개로 이루어진 배열 $A$와 정수 $K$가 주어진다. 다음 연산을 $0$회 이상 원하는 만큼 실행할 수 있다.</p>

<ul>
<li>배열에서 인접한 두 원소를 골라 교환한다. 단, 두 원소의 값의 차이가 $K$ 이하여야 한다.</li>
</ul>

<p>배열 $A$를 오름차순으로 정렬할 수 있는지 판별해 보자.</p>

### 입력 

 <p>첫째 줄에 정수 $N$과 $K$가 주어진다. $(1 \le N \le 200\, 000;0 \le K \le 10^{9})$</p>

<p>둘째 줄에 배열 $A$의 원소 $A_1, \cdots, A_N$이 공백으로 구분되어 주어진다. $(0 \le A_i \le 10^{9})$</p>

### 출력 

 <p>정렬할 수 있다면 <span style="color:#e74c3c;"><code>YES</code></span>를, 그렇지 않다면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</p>

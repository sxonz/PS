### 문제 설명

<p>초콜릿을 좋아하는 정화는 $N$개의 초콜릿을 모았다. 친구의 건강을 염려한 은채는 초콜릿을 하나만 남기고 버리라고 했다. 정화는 눈물을 머금고 가지고 있는 초콜릿 중 몇 개를 골라 하나로 합치기로 했다.</p>

<p>초콜릿을 합치기 위해선 사용할 초콜릿들의 세로 길이가 모두 같아야 한다. 정화는 초콜릿의 세로 길이를 원하는 만큼 줄일 수 있다.</p>

<p>합친 후의 초콜릿 크기를 (세로 길이) $\times$ (가로 길이)라고 하자. 합친 후의 초콜릿 크기의 최댓값을 알아보자!</p>

### 입력 

 <p>첫째 줄에 초콜릿의 개수 $N$이 주어진다. ($1 \leq N \leq 200\,000$)</p>

<p>둘째 줄에 $1, 2, \cdots, N$번 초콜릿의 세로 길이를 나타내는 $N$개의 정수 $H_1, H_2, \cdots, H_N$이 공백으로 구분되어 주어진다. ($1 \leq H_i \leq 200\,000$)</p>

<p>셋째 줄에 $1, 2, \cdots, N$번 초콜릿의 가로 길이를 나타내는 $N$개의 정수 $W_1, W_2, \cdots, W_N$이 공백으로 구분되어 주어진다. ($1 \leq W_i \leq 200\,000$)</p>

### 출력 

 <p>합친 후의 초콜릿의 크기의 최댓값을 출력한다.</p>
